# of-task-extractor

## Requeriments
### pipenv
```
sudo pip3 install pipenv
```

### Intall dependencies
```
pipenv install
```

## Run
### Default arguments
```
pipenv run python app.py
```

### Arguments
#### detail
```
pipenv run python app.py <task> <path-project>
```

#### Example
```
pipenv run python app.py 1234 ~/projects/my-project
```

## Contribute
### Intall dev dependencies
```
pipenv install --dev
```
