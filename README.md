# of-task-extractor

## Requeriments
### Download the project
```
git clone https://github.com/starlone/of-task-extractor.git
cd of-task-extractor
```

### pipenv
```
sudo pip3 install pipenv
```

### Intall dependencies
```
pipenv install
```

## Run

```
pipenv run python main.py
```

## Contribute
### Intall dev dependencies
```
pipenv install --dev
```
